
# Mathematical Tools for Neuroscience (Neurobio 212 at Harvard)

*Developed by Ella Batty*

This repository contains all the materials for the course "Mathematical Tools for Neuroscience" (Neurobio 212) taught at Harvard University. The course is designed to provide graduate students with essential mathematical skills in quantitative modeling and data analysis relevant to neuroscience research. Materials are available at https://calebweinreb.github.io/MathTools/intro.html#
